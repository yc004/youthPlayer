# 校园电视播放系统

## 系统简介

校园电视播放系统是一个基于Python开发的视频播放系统，支持按照设置好的时间表播放本地视频、NAS视频和网络直播内容。系统具有以下特点：

- 支持多种视频格式播放
- 支持多屏幕管理（Windows 7）
- 内置看门狗程序，确保系统稳定运行
- 提供Web管理界面，支持远程监控和控制
- 支持多账户管理，适合团队协作

## 技术栈

- **后端**：Python 3.8+, Flask, SQLAlchemy
- **播放器**：VLC (libvlc)
- **定时任务**：APScheduler
- **屏幕管理**：pywin32
- **数据库**：SQLite

## 系统要求

- Windows 7操作系统
- Python 3.8+
- VLC播放器
- 至少2GB内存
- 足够的存储空间
- 网络连接（用于NAS和网络直播）

## 安装步骤

### 1. 创建虚拟环境

```bash
# 使用conda创建虚拟环境
conda create -n youthplayer python=3.8
conda activate youthplayer
```

### 2. 安装依赖

```bash
pip install -r requirements.txt
```

### 3. 安装VLC播放器

从[VLC官网](https://www.videolan.org/vlc/)下载并安装VLC播放器。

### 4. 配置系统

编辑`config.py`文件，根据实际情况修改配置参数：

- `VLC_PATH`：VLC播放器的安装路径
- `WEB_PORT`：Web服务端口
- `SECRET_KEY`：用于Flask会话加密的密钥

### 5. 启动系统

```bash
python main.py
```

## 使用方法

### 1. 登录系统

打开浏览器，访问 `http://localhost:5000/login`，使用默认管理员账户登录：

- 用户名：admin
- 密码：admin123

### 2. 管理播放时间表

在主页上，您可以：

- 添加新的播放时间表
- 编辑现有时间表
- 删除不需要的时间表
- 查看当前播放状态

### 3. 控制播放

您可以通过Web界面控制播放：

- 开始播放
- 停止播放
- 暂停播放
- 恢复播放

### 4. 管理用户

管理员可以在"用户管理"页面添加和删除用户，设置用户权限。

## 系统架构

### 核心模块

1. **播放器模块**：负责视频播放，支持本地、NAS和网络直播
2. **控制模块**：管理播放时间表，控制播放状态
3. **安全模块**：保证播放窗口不受干扰，实现看门狗功能
4. **Web管理界面**：远程监控和控制播放系统
5. **数据存储**：存储时间表、用户信息等数据

### 系统流程

1. 系统启动时加载配置文件
2. 初始化播放模块和控制模块
3. 启动Web服务
4. 启动看门狗程序
5. 根据时间表触发播放任务
6. 在指定屏幕上播放内容
7. 监控播放状态，确保系统稳定运行

## 注意事项

1. 确保VLC播放器已正确安装并配置路径
2. 确保网络连接正常，特别是对于NAS和网络直播内容
3. 定期备份数据库，防止数据丢失
4. 对于重要的播放任务，建议提前测试播放内容

## 故障排查

### 常见问题

1. **播放器无法启动**：检查VLC播放器是否正确安装，路径是否配置正确
2. **网络直播无法播放**：检查网络连接，确保直播URL有效
3. **Web界面无法访问**：检查Web服务端口是否被占用，防火墙是否允许访问
4. **时间表不执行**：检查系统时间是否正确，时间表设置是否合理

### 日志查看

系统日志存储在`playback_system.log`文件中，可以通过查看日志了解系统运行状态和错误信息。
